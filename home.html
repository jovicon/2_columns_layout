This is an example for a simple 2 column CSS layout build.

Using Float


We need to created a main "section" for our website, in this case we put one menu "aside" and one main "article".

inside our "section",tags "aside" and "article" must be using the style "float:left" to created two columns.

****IMPORTANT****


Use "overflow: hidden" is our touché when we build a layout with floats because this let us size our "section" as we want.

What does "overflow" do: specifies what happens if content overflows an element's box.

What does "hidden" attribute do: The overflow is clipped, and the rest of the content will be invisible.
